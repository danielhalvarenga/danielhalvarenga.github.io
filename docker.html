Docker

Introdução

Podemos definir o docker como uma tecnologia de Virtualização. Porém não como uma tecnologia de virtualização comum, onde se utilizam máquinas virtuais.

O docker é uma Engine de administração de containers. O mesmo é responsável pelo gerenciamento de serviços utilizando o linux LXC, sendo assim, os containers são startados em máquinas linux.

  • Open Source, escrito em Go(Linguagem criada pelo Google);
  • Empacota Software com vários níveis de isolamento(Você pode isolar configurações, bibliotecas, etc).
